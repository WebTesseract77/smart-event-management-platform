# EventSphere - Smart Event Management Platform 🌐

A full-stack Event Management Platform built with FastAPI, Next.js, and SQLite.

---

## 🚀 Key Features
* **Role-Based Workflows:** User, Organizer, and Admin portals.
* **Smart Engines:** QR-based digital passes and real-time attendance tracking.
* **Scalability:** Individual and team registration support.

---

## ⚙️ Installation & Setup
1. **Clone:** `git clone https://github.com/WebTesseract77/smart-event-management-platform.git`
2. **Backend:** Setup `venv`, install `requirements.txt`, and run `uvicorn app.main:app --reload`.
3. **Frontend:** Run `npm install` and `npm run dev`.
